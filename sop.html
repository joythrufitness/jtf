Website Editor — How-To Guide

A step-by-step guide for updating the JTF website

Desktop Recommended This guide is best used on a desktop or laptop. The admin panel requires a larger screen to work properly.

This guide walks you through how to edit website content, download your changes, and publish them to the live site. No technical experience needed — just follow the steps below.

Part 1 — Open the Editor
1

Go to the Editor

Open your browser and go to the private editor link you were given. Bookmark it for easy access.

⚠ Keep this link private. Do not share it publicly.
2

The Editor Opens

You'll see the Site Content Editor with a list of sections on the left sidebar — Hero, Mission, Personal Training, Trainers, and more. Each section controls a different part of the website.

Part 2 — Make Your Changes
3

Click a Section to Open It

Click any section header (e.g. "Trainers" or "Testimonials") to expand it. Click again to collapse it. Use the Expand All / Collapse All button at the top right to open or close everything at once.

4

Edit Text Fields

Click into any text box or text area and type your changes. All sections work the same way — just click and edit.

5

Update an Image

Next to any image field, click Browse… to pick a photo from your computer. A preview will appear so you can confirm it looks right.

  • The file name will fill in automatically
  • You will also need to upload the actual image file to the images/ folder on GitHub (covered in Part 3)
6

Add or Remove a Trainer / Testimonial

In the Trainers or Testimonials sections, use the buttons at the bottom to add a new entry. To remove one, click the red ✕ Remove button on that block.

💾
Auto-save is on. Your work is automatically saved in your browser every few seconds. If you accidentally close the tab, a "Restore" banner will appear next time you open the editor — click it to get your work back.
Part 3 — Download Your Changes
7

Click "Download index.html"

When you're done editing, click the red ⬇ Download index.html button — either at the top right of the page or at the bottom of the left sidebar. You can also press Cmd + S on a Mac or Ctrl + S on Windows.

A file called index.html will download to your computer (usually in your Downloads folder).

Part 4 — Publish to the Live Site
8

Go to GitHub

Open a new browser tab and go to github.com/joythrufitness/jtf. Sign in if prompted.

9

Upload the File

On the repository page, click Add file → Upload files.

  • Drag your downloaded index.html file into the upload box, or click choose your files to find it
  • GitHub will show it in the list
10

Commit the Changes

Scroll down to the Commit changes section. Leave the message as-is or type a short note like "Update trainer info". Then click the green Commit changes button.

11

Wait ~60 Seconds

The live website at joythrufitness.com will update automatically. Wait about a minute, then refresh the page to see your changes.

💡 If you don't see your changes, try a hard refresh: Cmd + Shift + R on Mac, or Ctrl + Shift + R on Windows.
🖼
Uploading a new image? Do a second upload — same steps as above, but this time go into the images/ folder first, then upload your image file there. The website looks for images in that folder by name.
